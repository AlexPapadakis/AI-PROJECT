Σχολιασμός Αποτελεσμάτων

Επίδραση Αριθμού Κανόνων

Λιγότεροι κανόνες είναι καλύτεροι: Τα μοντέλα με μεγάλο radius (0.8) που παράγουν λιγότερους κανόνες επιτυγχάνουν καλύτερη απόδοση κατά ~16.4% σε σχέση με τα μοντέλα με μικρό radius (0.2).

Λόγος: Λιγότεροι κανόνες → Καλύτερη γενίκευση → Αποφυγή υπερεκπαίδευσης

Class-Dependent vs Class-Independent Clustering

Class-Dependent είναι καλύτερο: Η ξεχωριστή ομαδοποίηση ανά κλάση επιτυγχάνει καλύτερη απόδοση κατά ~8.2%.

Λόγος: Καθαρότερα clusters ανά κλάση → Πιο ερμηνεύσιμοι κανόνες → Καλύτερη αναγνώριση patterns

Επικάλυψη Ασαφών Συνόλων

Η επικάλυψη των ασαφών συνόλων στις εισόδους επιτρέπει την ταυτόχρονη ενεργοποίηση πολλαπλών κανόνων, βελτιώνοντας την ομαλότητα του μοντέλου και τη γενίκευση.

Προτάσεις Βελτίωσης

  • Cost-sensitive learning: Μεγαλύτερο κόστος για False Negatives σε ιατρικές εφαρμογές
  • Feature engineering: Δημιουργία κατηγοριών ηλικίας και risk groups
  • Ensemble methods: Συνδυασμός καλύτερων μοντέλων
  • Cross-validation: Πιο αξιόπιστη αξιολόγηση με k-fold CV

2ο Μέρος - Epileptic Seizure Recognition Dataset

Πληροφορίες Dataset
  • Πηγή: UCI Repository - Epileptic Seizure Recognition
  • Δείγματα: 11,500 instances
  • Χαρακτηριστικά: 179 features
  • Κλάσεις: 5 (Epileptic seizure activity, Tumor area, Healthy area, etc.)
  • Διαχωρισμός: 60% training, 20% validation, 20% test

Μεθοδολογία Grid Search

Λόγω της υψηλής διαστασιμότητας, εφαρμόστηκε συστηματική προσέγγιση με:

Παράμετροι Grid Search
  • Feature Selection: Relief algorithm με 5 nearest neighbors
  • Αριθμός Χαρακτηριστικών: [15, 25, 35]
  • Radius Subtractive Clustering: [0.3, 0.5, 0.7]
  • Cross-Validation: 5-fold stratified
  • Συνολικοί Συνδυασμοί: 3 × 3 = 9

Αποτελέσματα Grid Search

Βέλτιστες Παράμετροι

Μετά από εκτενή αναζήτηση, οι βέλτιστες παράμετροι είναι:

  • Αριθμός Χαρακτηριστικών: 25
  • Radius: 0.5
  • Cross-Validation Error: 0.125 (87.5% accuracy)

Σύγκριση με Grid Partitioning

Subtractive Clustering vs Grid Partitioning:

  • Τρέχον μοντέλο: 45 κανόνες με 25 features
  • Grid με 2 MFs/είσοδο: 2²⁵ = 33,554,432 κανόνες
  • Grid με 3 MFs/είσοδο: 3²⁵ = 847,288,609,443 κανόνες

Συμπέρασμα: Το Subtractive Clustering είναι απολύτως απαραίτητο για υψηλή διαστασιμότητα!

Ανάλυση Χαρακτηριστικών

Τα χαρακτηριστικά που επιλέχθηκαν από τον Relief algorithm αντιπροσωπεύουν:

  • Συχνοτικές συνιστώσες με υψηλή διακριτική ικανότητα
  • Temporal patterns σχετικά με epileptic activity
  • Statistical features που διαχωρίζουν τις κλάσεις

Συμπεράσματα 2ου Μέρους

Η συστηματική προσέγγιση με Grid Search επέτρεψε την εύρεση βέλτιστων παραμέτρων που επιτυγχάνουν:

  • Υψηλή ακρίβεια: 89.2% στο test set
  • Μείωση διαστασιμότητας: Από 179 σε 25 features (86% μείωση)
  • Αποδοτικότητα: 45 κανόνες αντί για εκατομμύρια
  • Ερμηνευσιμότητα: Class-dependent clustering για καθαρότερα patterns

Γενικά Συμπεράσματα

Βασικά Ευρήματα

  1. Αριθμός Κανόνων: Λιγότεροι κανόνες συνήθως προσφέρουν καλύτερη γενίκευση
  2. Class-Dependent Clustering: Ξεχωριστή ομαδοποίηση ανά κλάση βελτιώνει την απόδοση
  3. Feature Selection: Κρίσιμη για υψηλή διαστασιμότητα
  4. Grid Search: Απαραίτητη για βελτιστοποίηση παραμέτρων
  5. Cross-Validation: Εξασφαλίζει αξιόπιστη αξιολόγηση

Πρακτικές Συστάσεις

  • Χρήση Subtractive Clustering για μείωση πολυπλοκότητας
  • Εφαρμογή feature selection για υψηλή διαστασιμότητα
  • Class-dependent clustering για καλύτερη ερμηνευσιμότητα
  • Προσεκτική επιλογή μετρικών αξιολόγησης (ειδικά για ιατρικές εφαρμογές)
  • Συστηματική βελτιστοποίηση παραμέτρων με grid search
